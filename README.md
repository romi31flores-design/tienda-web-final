# Tienda Web Final

Proyecto final de Programación III.

## Descripción
Sitio web de una tienda online desarrollado con HTML, CSS y JavaScript.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- GitHub Pages

## Cómo correr el proyecto
Abrir el archivo `index.html` en el navegador o ingresar al sitio publicado mediante GitHub Pages.
